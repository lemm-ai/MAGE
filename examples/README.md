# Examples README

This directory contains example scripts demonstrating various MAGE features.

## Running Examples

### Basic Example

Simple audio generation with default settings:

```bash
python examples/basic_example.py
```

Generates a 10-second ambient track.

### Advanced Example

Demonstrates advanced features including:
- Custom configuration
- Multiple styles
- Audio effects
- Batch generation

```bash
python examples/advanced_example.py
```

Generates three variations with different styles and tempos.

### Configuration Example

Shows how to use YAML configuration files:

```bash
python examples/config_example.py
```

Creates and uses a custom configuration file.

## Output

Generated audio files are saved to `examples/output/`.

## Customization

Feel free to modify these examples to experiment with different:
- Durations
- Styles (ambient, electronic, orchestral, jazz, rock, classical)
- Tempos (20-300 BPM)
- Keys
- Complexity levels
- Effects (reverb, compression, EQ)
